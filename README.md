# Music Player - Web

> Streaming Music Players for the Web


## Building the website

```
$ git clone https://github.com/musicplayer-io/musicplayer-io.git
$ cd musicplayer-io

// Build CSS
$ npm install
$ npm install -g gulp
$ gulp

// Build HTML
$ npm install -g markade
$ markade compile
$ markade server
```
